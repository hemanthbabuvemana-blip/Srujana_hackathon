ACTMS — Unified Design System & Page Spec

(Styled only in black, gray, white, frosted glass, radiant beams)

⸻

Global Theme & System
	•	Background
	•	Base: #000000 → #0A0A0A gradient
	•	Radiant beams: diagonal soft white/gray streaks with blur + subtle animation
	•	Always present across all pages
	•	Typography
	•	Font: Inter / SF Pro / Poppins (sans-serif, modern)
	•	Headline: Bold, white (#FFFFFF)
	•	Body: Regular/Medium, gray (#B3B3B3)
	•	Glass Panels (cards, nav, modals, forms, chatbot)
	•	Fill: rgba(255, 255, 255, 0.04)
	•	Border: 1px solid rgba(255, 255, 255, 0.08)
	•	Blur: backdrop-filter: blur(14px)
	•	Radius: 16px
	•	Shadow: 0 6px 30px rgba(0,0,0,0.6)
	•	Buttons
	•	Primary CTA: White background (#FFFFFF), black text (#000000), radius 9999px, hover → invert (black bg, white text).
	•	Secondary CTA: Glass panel style, border, white text, hover → subtle white glow.
	•	Animations
	•	Sections fade-in on scroll
	•	Cards hover → soft glow outline
	•	Tabs → slide-in transitions
	•	Alerts / Chat messages → slide from bottom

 Dashboard Page

Purpose: High-level data overview
	•	Layout: Glass tab bar with 5 tabs → Overview, Financial, Security, Operational, Admin
	•	Charts inside Glass Cards:
	•	Overview: Pie (Tender Status), Bar (Suspicion Score)
	•	Financial: Bar (Tender Value Distribution)
	•	Security: Doughnut (Risk Levels: Safe / Suspicious — white & gray tones only, with opacity differences)
	•	Operational: Line (Activity per day)
	•	Admin: KPI cards → Total Tenders, Active Users, AI Alerts

(⚠️ Charts stay monochrome: white → light gray → dark gray gradients, no neon colors)

	•	Color Tokens
	•	Background: #000000 → #0A0A0A
